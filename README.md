


# Reactjs-Uploader
A flexible multi language React component for uploading files 
# (supported persian language and rtl direction) 

# Used Modules
<ul>
  <li>material-ui</li>
  <li>redux</li>
  <li>react-dropzone</li>
  <li>react-redux-multilingual</li>
  <li>jss</li>
  <li>axios</li>
  <li>mdi/font</li>
</ul>

## Getting Started
#### Installation
```bash
$ npm install
$ npm start
```
